Arcade help

Some tips and tricks to help you use the arcade machine.

First, turn the machine on at the plug. The first game will start automatically, but it could take some time, up to 5 minutes.

Once the game has started, you need to "insert coin". Use the left side-button to add as many credits as you like.

This is the player 1 start button. Some games allow 2 players. If you've added more than one credit, you can use the player 2 start button to the right of this.

For player 1, this is the button used by most games. Some games use 2, 3 or more of the buttons.

There are a few different games available, from classic, to more modern. If you'd like to try out a different game, press this button. The next game should load after a few seconds.

Inside the coin door is a small wireless keyboard with trackpad. This can be useful if you get stuck, or if the game crashes. For example you can use it to reboot Windows.

When you have finished playing, the simplest thing to do is turn off the machine at the plug. It's best to leave it switched of for 1-2 minutes before turning it on again.
